Basic understanding about git

git init: khoi tao git, bat dau su dung git

git status check file xem co trong repo(kho) chua, kiem tra file da duoc add chua, xanh la add roi con do la chua

git commit: the hien message

git branch: chon nhanh de day vao

git remote add origin + dia chi: xac dinh dia chi day len

git push de day code len
